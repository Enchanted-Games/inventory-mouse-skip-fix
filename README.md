# Inventory Mouse Skip Fix

Fixes the camera flicking that can sometimes happen when closing a screen (e.g. your inventory) and moving the mouse at the same time on macOS

## Config

You can adjust the amount of mouse inputs that are ignored after closing a screen from the config file. This is located at `.minecraft/config/eg_mouse_skip_fix.properties`, you will need to restart the game to apply changes
